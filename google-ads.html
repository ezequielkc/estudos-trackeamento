Google Ads e Tracking

Google Ads é a plataforma de publicidade do Google. Para profissionais de tracking, o foco está em rastrear conversões, medir ROI e integrar dados entre Google Ads, GTM e GA4.

Por que Tracking é Essencial no Google Ads?

Sem rastreamento adequado, você não sabe quais campanhas, palavras-chave ou anúncios estão gerando conversões reais. O tracking permite medir ROI, otimizar campanhas baseado em dados e atribuir conversões corretamente.

O que Você Precisa Rastrear:

Conversões do Site

  • Compras realizadas
  • Formulários enviados
  • Cadastros e assinaturas
  • Downloads
  • Visualizações de página-chave

Dados de Campanha

  • Cliques e impressões
  • Custo por clique (CPC)
  • Custo por conversão (CPA)
  • ROAS (Return on Ad Spend)
  • Atribuição de conversões

Comportamento do Usuário

  • Páginas visitadas
  • Tempo no site
  • Taxa de rejeição
  • Fluxo de navegação
  • Interações com elementos

Dados de Atribuição

  • Primeiro clique
  • Último clique
  • Modelo baseado em dados
  • Jornada completa do cliente
  • Touchpoints múltiplos

Rastreamento de Conversões no Google Ads

Rastrear conversões é fundamental para medir o sucesso das campanhas e otimizar investimentos. Existem várias formas de implementar tracking de conversões no Google Ads.

O que são Conversões no Google Ads?

Conversões são ações valiosas que usuários realizam após clicar em seus anúncios. Cada conversão pode ter um valor atribuído, permitindo calcular ROI e otimizar campanhas para maximizar receita.

Métodos de Rastreamento:

Método Como Funciona Quando Usar
Tag de Conversão do Google Ads Tag JavaScript específica do Google Ads instalada no site Quando não usa GTM ou precisa de tracking direto
Via Google Tag Manager Tag de conversão configurada no GTM e disparada por eventos Recomendado - Centraliza gerenciamento de tags
Importação do GA4 Conversões do GA4 importadas automaticamente para Google Ads Quando já usa GA4 e quer unificar dados
Conversões Offline Importação manual de conversões (ligações, visitas à loja) Para conversões que não acontecem online

Como Criar uma Ação de Conversão:

  1. Acesse o Google Ads e vá em Ferramentas e Configurações > Conversões
  2. Clique em "+ Nova ação de conversão"
  3. Escolha o tipo de conversão:
    • Site: Para ações no site (compras, formulários)
    • App: Para ações em aplicativos
    • Ligação: Para ligações telefônicas
    • Importação: Para importar de outras fontes
  4. Configure a conversão:
    • Nome da conversão
    • Categoria (compra, cadastro, etc.)
    • Valor (fixo ou variável)
    • Contagem (uma ou múltiplas)
    • Janela de atribuição
  5. Copie o ID de conversão (formato: AW-XXXXXXXXX)
  6. Configure a tag no GTM ou instale diretamente no site

Tipos de Conversões Comuns:

Micro-Conversões

  • Visualização de página-chave
  • Scroll de 75% da página
  • Tempo na página > 2 minutos
  • Download de material
  • Visualização de vídeo

Macro-Conversões

  • Compra realizada
  • Formulário enviado
  • Assinatura ativada
  • Agendamento confirmado
  • Lead qualificado

Integração Google Ads com GTM

Integrar Google Ads com Google Tag Manager é a melhor prática para gerenciar tags de conversão de forma centralizada e sem necessidade de alterar código.

Passo a Passo: Configurar Tag de Conversão no GTM

1. Obter ID de Conversão do Google Ads

No Google Ads:
1. Vá em Ferramentas > Conversões
2. Selecione ou crie uma ação de conversão
3. Copie o ID de conversão (ex: AW-123456789)
4. Anote também o Label de conversão (se houver)

2. Criar Tag no GTM

No Google Tag Manager:
1. Crie uma nova Tag
2. Tipo: "Google Ads: Conversion Tracking"
3. Configuration Tag: (opcional) Crie uma tag de configuração
4. Conversion ID: Cole o ID (AW-XXXXXXXXX)
5. Conversion Label: (se aplicável)
6. Conversion Value: {{Valor da Conversão}} (variável)
7. Order ID: {{ID da Transação}} (variável)
8. Currency Code: BRL (ou variável)

Configurar Acionador para Conversão:

Exemplo: Rastrear Envio de Formulário

  1. Crie um acionador do tipo "Envio de formulário"
  2. Configure condições (ex: ID do formulário = "contact-form")
  3. Associe o acionador à tag de conversão do Google Ads
  4. Teste usando Preview Mode do GTM
  5. Publique e valide no Google Ads (pode levar até 24h)

Enviar Dados de Conversão via DataLayer:

Exemplo: Compra com Valor e ID de Transação

// Quando uma compra é concluída
dataLayer.push({
    'event': 'purchase',
    'transaction_id': 'T12345',
    'transaction_total': 199.90,
    'transaction_currency': 'BRL',
    'transaction_items': [{
        'item_id': 'SKU123',
        'item_name': 'Produto',
        'price': 199.90,
        'quantity': 1
    }]
});

// No GTM, configure variáveis para capturar:
// - transaction_id → Order ID
// - transaction_total → Conversion Value
// - transaction_currency → Currency Code

Variáveis Úteis para Google Ads:

Integração Google Ads com GA4

Linking Google Ads com GA4 permite importar conversões do GA4 para o Google Ads e ter uma visão unificada da jornada do cliente.

Benefícios da Integração

Ao conectar Google Ads com GA4, você pode importar conversões do GA4, usar dados do GA4 para otimização de lances, e analisar a jornada completa do cliente desde o clique até a conversão.

Como Fazer o Linking:

  1. No GA4, vá em Administrador > Links do Google Ads
  2. Clique em "Link" e selecione sua conta do Google Ads
  3. Escolha quais propriedades do GA4 conectar
  4. Ative "Importar dados de conversão"
  5. Selecione quais eventos do GA4 importar como conversões:
    • purchase
    • generate_lead
    • sign_up
    • Ou eventos customizados marcados como conversão
  6. Configure atribuição (mesma do GA4 ou específica do Google Ads)
  7. Salve e aguarde até 24h para sincronização

Importar Conversões do GA4:

Configuração no GA4:

1. No GA4, marque eventos como conversões:
   - Vá em Administrador > Eventos
   - Encontre o evento (ex: purchase, form_submit)
   - Ative "Marcar como conversão"

2. No Google Ads, as conversões aparecerão automaticamente:
   - Vá em Ferramentas > Conversões
   - Procure por "GA4: [nome do evento]"
   - Use essas conversões para otimização

Vantagens da Integração GA4 + Google Ads:

Dados Unificados

Veja dados de tráfego do Google Ads no GA4 e conversões do GA4 no Google Ads, criando uma visão completa da performance.

Otimização Automática

Use dados do GA4 (como engajamento, tempo na página) para otimizar lances e estratégias de bid no Google Ads.

Atribuição Avançada

Aproveite modelos de atribuição do GA4 (baseado em dados, primeiro clique, último clique) para entender melhor a jornada do cliente.

Atribuição de Conversões no Google Ads

Atribuição determina qual clique ou interação recebe crédito por uma conversão. Entender modelos de atribuição é essencial para análise correta de ROI.

O que é Atribuição?

Quando um usuário clica em múltiplos anúncios antes de converter, a atribuição define qual clique recebe o crédito pela conversão. Diferentes modelos atribuem crédito de formas diferentes.

Modelos de Atribuição no Google Ads:

Modelo Como Funciona Quando Usar
Último Clique 100% do crédito vai para o último clique antes da conversão Padrão do Google Ads. Simples e direto.
Primeiro Clique 100% do crédito vai para o primeiro clique Para entender qual canal inicia a jornada
Baseado em Dados Usa machine learning para distribuir crédito baseado em padrões históricos Requer dados suficientes. Mais preciso.
Linear Crédito distribuído igualmente entre todos os cliques Para jornadas com múltiplos touchpoints
Decrescente Mais crédito para cliques mais recentes, menos para antigos Para valorizar interações recentes
Posição 40% primeiro, 40% último, 20% meio Para balancear primeiro e último toque

Janela de Lookback:

Configurando Janela de Atribuição

A janela de lookback define quanto tempo antes da conversão um clique pode ser atribuído. Opções comuns:

  • 30 dias: Padrão. Cliques até 30 dias antes contam
  • 60 dias: Para produtos com ciclo de compra longo
  • 90 dias: Para B2B ou produtos de alto valor
  • 1 dia: Para conversões rápidas (ex: downloads)

Rastreamento Cross-Device:

Conversões Cross-Device

O Google Ads pode rastrear quando um usuário clica no mobile mas converte no desktop (ou vice-versa) usando dados de login do Google.

Atribuição Multi-Touch

Com modelos de atribuição avançados, você pode ver todos os touchpoints na jornada do cliente, não apenas o último clique.

Eventos e Parâmetros para Tracking

Rastrear eventos específicos permite análise detalhada do comportamento do usuário e otimização mais precisa das campanhas.

Eventos Importantes para Rastrear:

Eventos de Conversão

  • purchase (compra)
  • generate_lead (lead)
  • sign_up (cadastro)
  • add_to_cart (carrinho)
  • begin_checkout (checkout)

Eventos de Engajamento

  • page_view (visualização)
  • scroll (rolagem)
  • video_start (vídeo)
  • file_download (download)
  • form_start (formulário iniciado)

Eventos de E-commerce

  • view_item (produto visto)
  • add_to_cart (adicionar carrinho)
  • remove_from_cart (remover)
  • purchase (compra)
  • refund (reembolso)

Parâmetros Essenciais

  • transaction_id
  • value (valor)
  • currency (moeda)
  • items (produtos)
  • campaign_id

Estrutura de Evento de Compra:

Exemplo Completo via DataLayer:

dataLayer.push({
    'event': 'purchase',
    'transaction_id': 'T12345',
    'value': 299.90,
    'currency': 'BRL',
    'items': [{
        'item_id': 'SKU123',
        'item_name': 'Produto Exemplo',
        'category': 'Eletrônicos',
        'quantity': 1,
        'price': 299.90
    }],
    'google_ads_conversion_id': 'AW-123456789',
    'google_ads_conversion_label': 'purchase_label'
});

Rastreamento de Parâmetros UTM:

Capturar Dados de Campanha

Use parâmetros UTM para rastrear origem do tráfego:

  • utm_source: Origem (google, facebook)
  • utm_medium: Meio (cpc, email)
  • utm_campaign: Nome da campanha
  • utm_term: Palavra-chave (para busca)
  • utm_content: Conteúdo específico do anúncio

No GTM, crie variáveis para capturar esses parâmetros e envie como parâmetros customizados para o Google Ads.

Boas Práticas de Tracking no Google Ads

Seguir boas práticas garante dados precisos e análises confiáveis para tomada de decisão.

Recomendações Essenciais:

Validação de Conversões

Sempre valide conversões usando Preview Mode do GTM e verifique no Google Ads se estão sendo registradas corretamente. Aguarde até 24h para aparecerem nos relatórios.

Evitar Duplicação

Não instale tag de conversão do Google Ads diretamente no site E via GTM. Escolha um método para evitar contagem dupla de conversões.

Valores de Conversão

Configure valores de conversão sempre que possível. Isso permite calcular ROI real e otimizar campanhas para maximizar receita, não apenas conversões.

Testes A/B de Tracking

Teste diferentes implementações de tracking para garantir que está capturando todos os eventos importantes. Use DebugView e Preview Mode regularmente.

Métricas de Tracking Importantes:

CPA (Cost Per Acquisition)

Custo por conversão. Métrica mais importante para medir eficiência das campanhas. Calcule: Custo Total / Número de Conversões.

ROAS (Return on Ad Spend)

Retorno sobre investimento em publicidade. Receita gerada dividida pelo custo das campanhas. ROAS de 4:1 significa R$4 de receita para cada R$1 gasto.

Taxa de Conversão

Percentual de cliques que resultam em conversão. Calcule: (Conversões / Cliques) × 100. Ajuda a identificar campanhas eficientes.

Valor por Conversão

Valor médio de cada conversão. Importante para entender qualidade das conversões, não apenas quantidade.
